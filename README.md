## 프로젝트 개요 (한국어)
- 이 저장소는 Liquid Glass 스타일(유리 같은 블러/글래스 효과)을 구현하는 예시 코드와 메모를 포함합니다.

## Project Overview (English)
- This repository contains example code and notes demonstrating a Liquid Glass style (glass-like blur/filter effects).
- See the `srcForLiquidGlassStyle/` folder for the full code and short examples.

## 구조 / Structure
- `srcForLiquidGlassStyle/Fullcode/` : 완성된 예시 컴포넌트 및 TypeScript(X) 파일
